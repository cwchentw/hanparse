# hanparse

A lightweight, rule-based Korean sentence parser written in TypeScript.

## Overview

`hanparse` is designed to parse Korean sentences using a deterministic, rule-driven approach.
It runs entirely on the client side, with no backend dependencies.

## Features

* Deterministic parsing behavior
* Fully client-side (no backend required)
* Simple and extensible rule system

## System Requirements

### Production

* Any modern browser with ES6 support

### Development

- Bun (for dependency management and tooling)
- Make (for running build, type check, and test tasks)

## Design Goals

* Deterministic behavior
* Zero backend dependency

## Non-Goals

* Morphological (stem) analysis
* Grammar checking or correction
